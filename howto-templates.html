How-To Guide & Text Templates

How to Locate and Turn on the Scanner

  1. Scanner is located in back office, next to desktop computer
  2. Press the "scan" button to turn on (blue light should illuminate)
  3. Desktop passkey: 6570

How to Scan

  1. Make sure scanner blue "scan" button is lit
  2. Align papers carefully and stack evenly
  3. No thicker than your pinky finger
  4. Face papers away from you, right side up
IMPORTANT: Remove ALL staples before scanning!

How to Scan Daily Notes

  1. Daily notes are located in top file cabinet drawer
  2. Separate into two sections:
    • Daily sheets
    • Massage SOAP notes
  3. Scan each section as separate files
  4. Name files as follows:
    • Chiro notes: BHAM CHIRO daily [DATE]
    • Massage notes: MASSAGE SOAP notes [DATE]
  5. Shred documents after scanning

How to Scan EOB's

  1. Open all envelopes, maintain original order
  2. Remove staples from all documents
  3. Scan in manageable sections
  4. Name files: BeWell EOB's [DATE]
  5. Separate any checks and paperclip together for Silvio's review
NOTE: Wait for Silvio's instructions before processing checks

General Scanning Tips

  • Always add current date to file names
  • Use the paper cutter for precise document handling
  • Double-check that all staples are removed
  • Review scanned documents for clarity before finalizing
