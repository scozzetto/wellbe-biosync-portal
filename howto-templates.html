How To's

How to Locate and Turn on the Scanner

1. Scanner is located in back office, next to desktop computer
2. Open paperholder at the bottom
3. Press the "scan" button to turn on (blue light should illuminate)
4. Turn on desktop - Passkey: 6570

How to Scan

1. Make sure scanner blue "scan" button is lit
2. Align papers carefully and stack evenly
3. No thicker than your pinky finger
4. NO STAPLES!!!
5. Face papers away from you, right side up
6. Hit scan
7. Follow pop-up instructions for continuing scan or finishing scan
8. Rename file with appropriate title (see below for titles)
9. Save to dropbox

How to Scan Daily Notes

1. Daily notes are located in top file cabinet drawer
2. Separate chiro daily notes from massage SOAP notes
3. Scan daily chiro sheets first as one file
Title: BHAM CHIRO daily [DATE]
4. Scan massage SOAP notes second as separate file
Title: MASSAGE SOAP notes [DATE]
5. Throw away (shred) after scanning

How to Scan EOB's

1. Open all envelopes, keep original order
2. Remove staples from all papers
3. Scan in sections (don't try to scan everything at once)
4. Title: BeWell EOB's [DATE]
5. If there are checks, separate them and paperclip together for Silvio to review
Wait for Silvio's instruction before doing anything with checks

General Scanning Guidelines

  • Always use current date in file names
  • Use paper cutter for clean document handling
  • Remove ALL staples before scanning
  • Scan in manageable sections if stack is too thick
  • Face all papers away from you, right side up
  • Save all scanned documents to Dropbox
  • Shred original documents after successful scanning (except checks)
